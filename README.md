## Description

### What is Redis?
Redis is an memory database. It creates a database stored in the memory.


## Getting Started

To clone this repo,

```bash 
git clone <url_of_repo> <name_of_your_dir-optional>
```
After run this on your pc by running

```bash
yarn 
yarn run dev
```
